About

This AI-powered agent automatically scans NASA's exoplanet databases on startup, specifically targeting planets discovered via the transit method. The system queries TESS, Kepler, K2, and confirmed planets datasets, classifies each discovery using ChatGPT AI, and stores them in a Supabase database for real-time viewing.

How it works:

  • Automatically scans NASA databases when server starts
  • Filters for transit method planets only
  • AI classification using ChatGPT (Confirmed/Candidate/False Positive)
  • Stores discoveries in Supabase database
  • Auto-refreshes every 10 seconds to show new planets
  • Displays planet details: name, host star, period, radius, classification

Data Sources:

  • TESS: Transiting Exoplanet Survey Satellite candidates
  • Kepler: Kepler Objects of Interest (KOIs)
  • K2: K2 Mission candidates
  • Confirmed: All confirmed transit method planets
